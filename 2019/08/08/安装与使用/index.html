Vue-devtools 安装

chrome应用商店安装

  1. 直接在线安装
  2. 离线安装 怎么在谷歌浏览器中安装.crx扩展

github 下载源码安装

  1. clone源码 地址：https://github.com/vuejs/vue-devtools

  2. 进入工程—>下载依赖—>编译

  3. 编译完成后，修改shells-chrome目录下的mainifest.json 中的persistant为true

  4. 打开谷歌浏览器的设置—>扩展程序，并勾选开发者模式,单击“加载解压扩展”，然后选择shells / chrome。
